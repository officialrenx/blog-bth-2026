Beranda

Hai semuanya, perkenalkan nama ku Bagas Tri Hadmodjo, biasa dipanggil Bagas. Saya lahir dan besar di kota Yogyakarta pada tanggal 15 Juni 1993. Saya anak ke 3 dari 4 bersaudara. Saya telah menempuh pendidikan S1 jurusan Sistem Informasi di STMIK AKAKOM Yogyakarta ( sekarang berganti nama menjadi UTDI ). Saat ini saya sedang berfokus di bidang programmer khususnya Front End. Framework yang sedang saya pelajari adalah Vue.js dan untuk Framework UI menggunakan Bootstrap. Saya sangat suka dengan dunia IT, apalagi seiring berkembangnya zaman, teknologi pasti selalu mengeluarkan sesuatu hal yang baru untuk kita pelajari. Intinya manusia di bumi jangan sampai dikendalikan oleh teknologi, malah justru kita yang harus mengendalikan mereka ini :D .


Oh ya keseharian saya adalah ngoding. Disamping ngoding, saya juga mempelajari teknologi - teknologi terbaru agar tidak ketinggalan zaman alias gaptek :D. Dan untuk hobi saya adalah bermain musik, seperti gitar, bass dan drum tapi terlalu condong ke Bass. Idola pemain Bass saya di dunia ini adalah Mike Dirnt ( Green Day ) dan Mark Hoppus ( blink182 ). Disamping bermain alat musik saya juga suka mendengarkan musik menggunakan aplikasi Youtube Music dan Spotify. Karena bagi saya, musik mewakili hidup saya dalam fase atau situasi apapun.
